> **요약**: 외국인 근로자가 검증된 아웃소싱 업체의 일자리를 안전하게 찾을 수 있는 구인구직 플랫폼

> **프로젝트**: 114Korea
> **버전**: 0.1.0
> **작성자**: acctb
> **날짜**: 2026-02-27
> **상태**: Draft

---

## 1. 개요

### 1.1 목적

외국인 근로자들이 사기나 불법 업체를 피하고, 검증된 아웃소싱 업체를 통해 안전한 일자리를 찾을 수 있도록 돕는 플랫폼을 구축한다.

### 1.2 배경

- 한국 내 외국인 근로자 수 지속 증가 (약 90만명+)
- 기존 114114kr.com, 114114korea.com 등의 플랫폼은 업체 검증 시스템이 미흡
- 외국인 근로자 대상 사기, 임금체불, 불법 파견 등의 문제 빈번
- 검증된 업체만 참여할 수 있는 신뢰 기반 플랫폼 부재

### 1.3 참고 사이트

- [114114kr.com](https://www.114114kr.com/) - 외국인/교포 구인구직 게시판
- [114114korea.com](https://114114korea.com/) - 구인/구직 정보 + 이력서 등록
- [KOWORK](https://kowork.kr/) - 외국인 특화 HR 플랫폼, 비자 정보 제공
